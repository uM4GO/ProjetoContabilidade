O projeto conta com um sistema que, conforme solicitado, registra um lançamento, lista os lançamentos, consulta saldo e filtra os lançamentos.
Todas as instruções de como utiliza-lo estão presentes no sistema.

Informações adicionais:
Como você decidiu estruturar sua solução?
Pensei em um sistema simples e objetiva conforme foi requisitado.

O que você melhoraria se tivesse mais tempo?
Persistência em banco de dados e interface web.

Quais dificuldades encontrou durante o desenvolvimento?
Vários while e ifs para validações de inputs.
